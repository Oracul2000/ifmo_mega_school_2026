Реализован интеллектуальный агент для проведения технических интервью на базе LangGraph и Mistral AI. Система имитирует поведение группы интервьюеров, анализирует полноту ответов в реальном времени, ведет скрытые логи размышлений (Internal Thoughts) и адаптирует стратегию опроса.
Структура проекта

Проект построен по модульному принципу, где каждый файл отвечает за конкретный этап жизненного цикла интервью:

defenitions.py: Общая схема данных (InterviewState). Описывает состояние системы: историю ходов, текущий прогресс, данные кандидата и накопленные мысли агентов.

question_planner.py: Модуль стратегического планирования. Анализирует резюме и формирует список глубоких технических вопросов, разделенных по темам.

critic.py: Сердце системы. Содержит логику двух виртуальных ролей:

main.py: Оркестратор. Собирает узлы в граф (StateGraph), управляет переходами между вопросами и формирует итоговый JSON-лог интервью.

evaluator.py
